# Reflective Functioning Dataset- Digital Health (2026)
This repository contains a single xlsx file with 3 different sheets containing the data that was used in the preprint: https://www.medrxiv.org/content/10.64898/2026.03.17.26348630v1
1. Sheet Name: Main Sample Data
   Contains the data used in main analyses. Includes Somascape-derived measures of somatic symptoms, demographic data and paper-based questionnaire data.
2. Sheet Name: Subsample BRFQ
   Contains the data used to analyze the test-retest reliability of the BRFQ-9. _t1/_t2 suffixes indicate time.
3. Sheet Name: Subsample RFQ
   Contains the data used to analyze the test-retest reliability of the RFQ-8. _t1/_t2 suffixes indicate time.
